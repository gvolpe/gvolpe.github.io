# website

Personal blog and website built with [Hugo](https://gohugo.io/) and a modified version of the [Loficode Theme](https://github.com/raisingpixels/loficode-hugo-theme).

See it live at [gvolpe.com](https://gvolpe.com).

## Run it locally

```console
$ nix develop # or better use echo 'use flake' > .direnv && direnv allow
$ hugo server
# ....................

Press Ctrl+C to stop
```
